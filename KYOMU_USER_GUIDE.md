# Kyomu Provider - User Guide

## What is Kyomu?

Kyomu is a new streaming provider that has been added to your anime streaming platform. It provides an alternative source for watching anime episodes, giving you more options if one provider is unavailable or slow.

## How to Use Kyomu

### On the Anime Info Page

1. Navigate to any anime's info page
2. Scroll down to the "Episodes" section
3. Look for the provider dropdown (usually shows "HiAnime" by default)
4. Click the dropdown and select **"Kyomu"**
5. The episode list will update to show Kyomu's available episodes
6. Click any episode to start watching

### On the Watch Page

1. While watching an anime episode
2. Look for the provider buttons below the video player
3. You'll see buttons labeled with provider names (e.g., "HiAnime", "Kyomu")
4. Click the **"Kyomu"** button to switch to that provider
5. The video will reload with Kyomu's stream

## Troubleshooting

### Episode Not Loading?
1. Try switching to a different provider (HiAnime or Kyomu)
2. Refresh the page
3. Check your internet connection

### Provider Not Showing?
- Some anime may not be available on all providers
- If Kyomu doesn't appear, the anime might not be in their database
- Try using HiAnime instead

### Video Quality Issues?
- Kyomu streams are typically high quality
- If you experience buffering, try:
  - Pausing for a few seconds to let it buffer
  - Switching to HiAnime
  - Checking your internet speed

## FAQ

**Q: Which provider is better?**
A: Both are good! Try both and see which works better for you.

**Q: Can I use both providers for the same anime?**
A: Yes! You can switch between them anytime.

**Q: Will my watch history work with both?**
A: Yes, your progress is tracked regardless of provider.

**Q: Why add another provider?**
A: Having multiple providers ensures you always have a working stream, even if one provider is down or slow.

**Q: Does Kyomu support all anime?**
A: Most anime are available, but coverage may vary. If an anime isn't on Kyomu, use HiAnime.
